# Steps to setup:
1. Go to the three dots on the top right of Chrome
2. Click on extentions-> Manage extentions
3. Enable developer mode
4. Select "Load unpacked"
5. When the downloaded repo is completed, go to the unzip directory and ensure all files are in the folder
6. the Load unpacked button in chrome will prompt you for you the location of the folder you are loooking to install select the unzipped folder
7. Then BOOM you have just added the extention.
